# 基本

APIへのリクエストパスは以下の通りです。

```
https://api.munesky.net/v1/
```

# レスポンス形式

基本的に以下のレスポンスが返却されます。

```
{
  message : "基本的に文字列で返却されます。",
  data : {
    データ(GETのみ)
  }  
}
```

# ステータス

基本的に`HTTP StatusCode`での判別になります。

| コード | 状態 | 詳細 |
| ---- | ---- | ----| 
| 200 | 成功 | |
| 404 | 失敗 | 存在しない |
| 400 | 失敗 | すでに存在 |
| 502 | 失敗 | サーバーエラー |

:::note 要望があればステータスコードを別で作ります。
