ユーザーのAFK状態を取得します。

::warn 認証が必要です。

# リクエスト
```
GET /v1/afk/:id
```

::warn レートリミットがかかっています。1つのIPにつき、`20/minute`です。

# レスポンス (例)
```
{
    "id" : "000000000000000000",
    "reason": "おやすみじかん",
    "time": "2024-1-1 00:00:00.000000"
}
```

| 値 | 型 | 詳細 |
| ---- | ---- | ---- |
| id | Snowflake(BigInt) | ユーザーのDiscordIDです。 |
| reason | String | AFKの理由です。 |
| time | datetime | AFK開始時刻です。 |
