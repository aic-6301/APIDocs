ユーザーのAFK状態を作成します。

:::warning 認証が必要です。 :::

# リクエスト
```
POST /v1/afk
```

:::warning レートリミットがかかっています。1つのIPにつき、`10/minute`です。 :::

# リクエストBody
```json
{
    "id": "ユーザーIDです。",
    "reason": "理由です。"
}
```

| 値 | 型 | 説明 |
| ---- | ---- | ---- |
| id | Snowflake(BigInt) | ユーザーのDiscordIDです。 |
| reason | String | AFKの理由です。 |

# レスポンス

## エラー

`HTTP StatusCode`:400

```json
{
    "message": "エラー内容"
}
```

`HTTP StatusCode`:429
```json
{
    "message": "Too Many Requests"
}
```

## エラーなし

`HTTP StatusCode`:200

```json
{
    "message": "OK"
}
```
