ユーザーのAFK状態を作成します。

::warn 認証が必要です。

# リクエスト
```
DELETE /v1/afk
```

::warn レートリミットがかかっています。1つのIPにつき、`10/minute`です。

# リクエストBody
```json
{
    "id": "ユーザーIDです。"
}
```

# レスポンス

## エラー

`HTTP StatusCode`:404

```json
{
    "message": "Not Found"
}
```

`HTTP StatusCode`:429
```json
{
    "message": "Too Many Requests"
}
```

## エラーなし

`HTTP StatusCode`:200

```json
{
    "message": "OK"
}
```
