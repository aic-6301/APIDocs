ユーザーのAFK状態を作成します。

:::warning
認証が必要です。 
:::

# リクエスト
```
DELETE /v1/afk
```

:::warnin
 レートリミットがかかっています。1つのIPにつき、`10/minute`です。
 :::

# リクエストBody
```json
{
    "id": "ユーザーIDです。"
}
```

# レスポンス(例)

## エラー

`HTTP StatusCode`:404

```json
{
    "message": "Not Found"
}
```

`HTTP StatusCode`:429
```json
{
    "message": "Too Many Requests"
}
```

## エラーなし

`HTTP StatusCode`:200

```json
{
    "message": "OK",
    "data": {
        "id" : "000000000000000000",
        "reason": "おやすみじかん",
        "time": "2024-1-1 00:00:00.000000"
        }
}
```
